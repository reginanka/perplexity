# 🖥️ Що таке Perplexity Computer

> Perplexity Computer — це AI-агент нового покоління, який може самостійно виконувати складні багатокрокові задачі: переглядати браузер, читати й писати файли, запускати код та координувати під-агентів.

---

## Ключова ідея

На відміну від звичайного Perplexity Search (який відповідає на питання), **Perplexity Computer діє** — він бере задачу і виконує її кроками, як людина за комп'ютером.

```
Користувач → задає мету → Computer планує кроки → виконує → повертає результат
```

---

## Як це працює

1. **Отримує задачу** від користувача (наприклад: "Знайди топ-10 SaaS інструментів для автоматизації і склади порівняльну таблицю")
2. **Розбиває** на підзадачі (пошук, читання сторінок, аналіз, форматування)
3. **Виконує кроки** — використовує браузер, код, файли
4. **Координує під-агентів**, якщо задача потребує паралельного виконання
5. **Повертає готовий результат** — звіт, файл, таблицю тощо

---

## Чим відрізняється від звичайного пошуку

| Звичайний Perplexity | Perplexity Computer |
|---|---|
| Відповідає на запит | Виконує задачу |
| Пасивний (чекає питання) | Активний (діє самостійно) |
| Результат — текст | Результат — файл, таблиця, код |
| Один крок | Багатокрокове виконання |
| Немає пам'яті між кроками | Зберігає контекст задачі |

---

## Де знайти

Perplexity Computer доступний через:
- Веб-інтерфейс [perplexity.ai](https://perplexity.ai) — розділ **Labs** або **Computer**
- API (в розробці / обмежений доступ)

> ⚠️ **Примітка:** На момент квітня 2026 Perplexity Computer знаходиться у стадії активного розвитку. Функціонал може змінюватися.
