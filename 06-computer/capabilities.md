# ⚙️ Можливості Perplexity Computer

> Що саме вміє робити Perplexity Computer — детальний огляд усіх інструментів і здібностей агента.

---

## 🌐 Браузер (Web Browsing)

Agent може самостійно:
- Відкривати та читати веб-сторінки
- Переходити по посиланнях
- Заповнювати форми (де дозволено)
- Робити скріншоти сторінок
- Збирати дані з кількох джерел одночасно

```
Приклад: "Зайди на сайт компанії X та витягни всі контактні emails"
```

---

## 📁 Робота з файлами

- Читати завантажені файли (PDF, DOCX, CSV, TXT)
- Створювати файли з результатами (звіти, таблиці)
- Обробляти великі документи по частинах
- Аналізувати зображення та скріншоти

---

## 💻 Виконання коду

- Запускати Python скрипти для обробки даних
- Проводити розрахунки та аналіз
- Генерувати графіки та візуалізації
- Парсити структуровані дані (JSON, XML, CSV)

```python
# Computer може виконати такий код сам:
import pandas as pd
df = pd.read_csv('data.csv')
print(df.describe())
```

---

## 🤖 Під-агенти (Sub-agents)

Для складних задач Computer може запускати **паралельних під-агентів**:
- Кожен агент виконує свою підзадачу
- Результати збираються і синтезуються
- Значно прискорює великі дослідження

```
Головний агент
├── Агент 1: досліджує конкурента A
├── Агент 2: досліджує конкурента B  
└── Агент 3: досліджує конкурента C
→ Синтез у фінальний звіт
```

## ⚠️ Обмеження

| Можливість | Статус |
|---|---|
| Авторизація на сайтах | ❌ Не підтримується |
| Платежі та транзакції | ❌ Не підтримується |
| Постійна пам'ять між сесіями | ⚠️ Обмежено |
| Реальний час виконання | ⏱️ Задачі можуть тривати хвилини |
| Складні UI взаємодії | ⚠️ Частково |
