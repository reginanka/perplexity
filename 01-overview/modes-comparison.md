# ⚙️ Режими Perplexity: Auto, Pro Search, Deep Research, Labs

> Більшість користувачів застряє на Auto Mode і не знає, що існують набагато потужніші режими.

---

## Огляд режимів

| Режим | Доступ | Швидкість | Глибина | Коли використовувати |
|---|---|---|---|---|
| **Auto** | Free + Pro | ⚡ Дуже швидко | ⭐⭐ | Прості питання, факти |
| **Pro Search** | Pro | 🔄 Середньо | ⭐⭐⭐⭐ | Складні питання, аналіз |
| **Deep Research** | Pro | 🐢 2-5 хвилин | ⭐⭐⭐⭐⭐ | Дослідження, звіти |
| **Labs** | Pro | Різна | Різна | Тестування нових функцій |

---

## Auto Mode

Perplexity сам вирішує — зробити простий пошук чи глибший аналіз.

**Підходить для:**
- «Яка погода в Києві?»
- «Хто такий Ілон Маск?»
- «Курс долара зараз»

**Не підходить для:**
- Складного аналізу
- Порівняння кількох продуктів
- Дослідження ринку

---

## Pro Search

Робить кілька ітерацій пошуку: уточнює запит, шукає з різних кутів, синтезує результати.

**Підходить для:**
- «Порівняй фреймворки FastAPI та Django для мого проекту»
- «Які найкращі інструменти автоматизації у 2026?»
- «Як налаштувати n8n з Webhook?»

**Хак:** Якщо на Free плані — починай питання зі слів «детально поясни» або «зроби аналіз» — це підвищує шанс що Auto автоматично переключиться в глибший режим.

---

## Deep Research

Автоматично виконує десятки пошуків, читає сотні сторінок, генерує структурований звіт.

**Підходить для:**
- Аналізу конкурентів
- Дослідження ринку
- Технічних огляд
- Написання детальних звітів

**Як правильно формулювати запит:**
```
❌ Погано:  "Розкажи про ринок CRM"
✅ Добре:   "Зроби детальний аналіз ринку CRM-систем для малого бізнесу 
            у 2026 році: основні гравці, ціни, тренди, порівняння функцій"
```

**Час виконання:** від 2 до 10 хвилин залежно від складності.

---

## Focus режими (фільтр джерел)

Можна обмежити пошук конкретним типом джерел:

| Focus | Що шукає |
|---|---|
| 🌐 Web | Весь інтернет (за замовчуванням) |
| 🎓 Academic | Наукові статті (PubMed, arXiv тощо) |
| ▶️ YouTube | Відео + транскрипти |
| 🤖 Reddit | Думки реальних людей |
| 📰 News | Свіжі новини |
| 💻 Code (GitHub) | Код, репозиторії |

**Хак:** Для технічних питань використовуй Focus → Reddit — там часто є реальні кейси та лайфхаки від практиків, яких немає в офіційній документації.
