# 🗺️ Карта екосистеми Perplexity

> Perplexity — це не просто пошуковик. Це платформа з кількома продуктами, які можна використовувати разом.

---

## Продукти екосистеми

```
Perplexity AI
│
├── 🔍 Search (основний продукт)
│   ├── Auto Mode
│   ├── Pro Search
│   └── Focus режими (Web, Academic, YouTube, Reddit...)
│
├── 🧪 Deep Research
│   └── Автоматичний аналіз з десятками пошуків
│
├── 🗂️ Spaces
│   ├── Особистий простір
│   └── Командний простір з базою знань
│
├── 💻 Perplexity Computer
│   ├── Браузер-агент
│   ├── Sub-агенти для складних задач
│   └── Автономне виконання завдань
│
├── ⚡ API
│   ├── Search API
│   ├── Agent API
│   ├── Embeddings API
│   └── Sonar моделі
│
└── 🤖 Моделі (доступні в Pro)
    ├── Sonar (власна модель Perplexity)
    ├── GPT-5 / GPT-4o
    ├── Claude Sonnet / Opus
    ├── Gemini Pro / Flash
    └── Grok
```

---

## Де що використовувати

| Задача | Продукт |
|---|---|
| Швидке питання з актуальними даними | Search (Auto) |
| Глибокий аналіз теми / ринку | Deep Research |
| Корпоративна база знань | Spaces |
| Автоматизація задач в браузері | Computer |
| Інтеграція у свій застосунок | API (Sonar) |
| Порівняння відповідей різних моделей | Pro Search → вибір моделі |

---

## Екосистема по підписках

```
Free ──────────────────────────────────────────────────────
  Auto Search, базові Spaces, обмежений Pro Search

Pro ($20/міс) ─────────────────────────────────────────────
  + Необмежений Pro Search
  + Deep Research
  + Вибір моделі (GPT-5, Claude, Gemini, Grok)
  + Perplexity Computer
  + Розширені Spaces

API (окремо) ──────────────────────────────────────────────
  Sonar Small / Large / Pro / Reasoning
  Оплата за токени
```

---

## Ключові переваги платформи

- 🔄 **Один інтерфейс** — замість 5 різних AI-інструментів
- 📎 **Завжди актуальна інформація** — не залежить від дати навчання моделі
- 🔗 **Цитати** — кожне твердження підкріплено джерелом
- 🧩 **API для розробників** — легко інтегрувати у свої проекти
