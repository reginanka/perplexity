# 🎯 Оператори та трюки для точного пошуку

> Більшість користувачів пишуть запити як у Google. Але Perplexity розуміє природну мову — використовуй це.

---

## Структура ефективного запиту

```
[Контекст] + [Задача] + [Обмеження] + [Формат]

Приклад:
"Я Python розробник. Порівняй Celery та RQ для черг задач.
Фокус на простоті налаштування. Дай таблицю з плюсами/мінусами."
```

---

## Ключові оператори

| Оператор | Приклад | Що робить |
|---|---|---|
| `site:` | `site:github.com n8n telegram` | Шукає тільки на конкретному сайті |
| `filetype:` | `filetype:pdf python tutorial` | Шукає файли певного типу |
| `""` (лапки) | `"Claude API" pricing` | Точний збіг фрази |
| `-` (мінус) | `AI tools -ChatGPT` | Виключає слово з пошуку |
| `OR` | `n8n OR Make.com tutorial` | Пошук за одним або іншим |

---

## Шаблони запитів для розробників

**Порівняння технологій:**
```
"Порівняй [A] та [B] для [конкретного кейсу]. 
Врахуй: продуктивність, складність налаштування, ціна.
Дай рекомендацію для [мій контекст]."
```

**Debugging:**
```
"[мова програмування] помилка: [текст помилки]
Контекст: [що ти робив]
Що означає і як виправити?"
```

**Документація:**
```
"site:docs.python.org [функція або метод]"
```

**Свіжі новини по темі:**
```
"[тема] після [дата] site:reddit.com OR site:hackernews.com"
```

---

## Трюки для кращих результатів

**1. Давай контекст про себе:**
```
✅ "Я новачок у Docker. Поясни volumes простими словами."
❌ "Що таке Docker volumes?"
```

**2. Проси конкретний формат:**
```
✅ "Дай покрокову інструкцію з кодом"
✅ "Зроби порівняльну таблицю"
✅ "Поясни як для 5-річної дитини"
```

**3. Уточнюй часові рамки:**
```
✅ "Найкращі Python бібліотеки для RAG у 2026 році"
❌ "Найкращі Python бібліотеки для RAG"
```

**4. Задавай follow-up запитання:**
Piсля відповіді натисни на одне з suggested questions або задай своє уточнення — контекст зберігається в межах сесії.

---

## Що НЕ варто шукати в Perplexity

- Локальні сервіси (кафе поряд, розклад автобусів)
- Речі що потребують логіну (банківські рахунки, CRM)
- Дуже нішеві запити на рідкісних мовах
