# 🎯 Focus-режими: шукай там де треба

> Focus дозволяє обмежити пошук конкретним типом джерел. Правильний Focus = набагато точніші відповіді.

---

## Всі доступні Focus-режими

### 🌐 Web (за замовчуванням)
Шукає по всьому інтернету. Підходить для 80% запитів.

### 🎓 Academic
Обмежує пошук науковими базами: PubMed, arXiv, Semantic Scholar, Google Scholar.

**Коли використовувати:**
- Медичні або наукові питання
- Пошук досліджень і статистики
- Перевірка наукових тверджень

```
Приклад: "Ефективність інтервального голодування — наукові дослідження 2024-2026"
```

### ▶️ YouTube
Аналізує відео та їх транскрипти.

**Коли використовувати:**
- Шукаєш туторіал по конкретній темі
- Хочеш дізнатись що кажуть експерти у відео
- Потрібне резюме відео без перегляду

```
Приклад: "n8n webhook tutorial 2026" → отримаєш огляд найкращих відео
```

### 🤖 Reddit
Шукає по постах і коментарях Reddit.

**Коли використовувати:**
- Реальні відгуки про продукти/сервіси
- Нестандартні рішення проблем
- Досвід практиків (не маркетинг)
- Порівняння інструментів очима користувачів

```
Приклад: "best free VPS for telegram bot reddit" 
→ реальні відповіді від розробників, не рекламні статті
```

### 📰 News
Тільки новинні джерела, відсортовані за свіжістю.

**Коли використовувати:**
- Актуальні події
- Новини про компанії/продукти
- Оновлення технологій

### 💻 Code (GitHub)
Шукає по репозиторіях та коду на GitHub.

**Коли використовувати:**
- Приклади реалізації конкретного функціоналу
- Бібліотеки для задачі
- Сніпети коду

```
Приклад: "telegram bot aiogram inline keyboard example"
```

---

## Порівняння Focus-режимів

| Задача | Кращий Focus |
|---|---|
| Загальне питання | Web |
| Наукові факти / статистика | Academic |
| Думки реальних людей | Reddit |
| Актуальні новини | News |
| Туторіал / навчання | YouTube |
| Приклади коду | Code |
| Порівняння продуктів (чесне) | Reddit |
| Медицина / здоров'я | Academic |

---

## Хак: комбінуй Focus + Deep Research

Можна запустити Deep Research з Focus Academic для глибокого наукового аналізу:

```
Focus: Academic
Режим: Deep Research
Запит: "Порівняй ефективність різних архітектур RAG систем — останні дослідження"
```
Результат: структурований звіт з посиланнями виключно на наукові роботи.
