## 🆓 Що можна зробити безкоштовно

Безкоштовний план Perplexity — один із найщедріших серед усіх AI-інструментів. Ось що реально доступно:

### Необмежені стандартні пошуки
Звичайний пошук (Quick Search / Auto режим) — **без ліміту**. Це вже потужніше за більшість конкурентів.

### 5 Pro Search на день
Pro Search — це поглиблений пошук, що аналізує більше джерел. 5 запитів на день достатньо для випадкового користувача.

### Доступ до топ-моделей (обмежено)
GPT-5.4, Claude Sonnet, Gemini — безкоштовно, але з лімітами. Після вичерпання ліміту переключається на базову модель.

### 3 Spaces безкоштовно
Можна створити до 3 Spaces для зберігання документів і AI-пошуку по них.

---

## 💎 Що дає Pro план

### 1. 300 Pro Search на день
Найголовніше оновлення. Pro Search глибше аналізує запит, ставить уточнюючі питання, агрегує більше джерел.

### 2. Deep Research без лімітів*
*500 запитів/місяць. Deep Research будує повноцінний аналітичний звіт із десятків джерел — незамінно для бізнес-аналізу, ресерчу конкурентів, наукових оглядів.

### 3. Генерація зображень
FLUX та інші моделі прямо в чаті. Можна генерувати ілюстрації до своїх досліджень.

### 4. Perplexity Computer (Labs)
AI-агент, що може керувати браузером, виконувати задачі в інтернеті автономно. Поки в бета-версії.

### 5. API кредити $5/місяць
Входять у ціну Pro! Якщо ви розробник — це частково покриває витрати на Sonar API.

## 🎯 Кому варто платити за Pro

**✅ Варто брати Pro, якщо ви:**
- Дослідник або аналітик, що робить 10+ глибоких запитів на день
- Підприємець, що регулярно досліджує ринок і конкурентів
- Контент-мейкер, якому потрібні актуальні дані для матеріалів
- Розробник, що тестує Perplexity API (кредити $5 включено)
- Ведете команду або Spaces з великим об'ємом документів

**❌ Залишайтеся на Free, якщо:**
- Використовуєте Perplexity кілька разів на тиждень
- 5 Pro Search на день вам достатньо
- Не потрібна генерація зображень
- Не плануєте використовувати Deep Research регулярно

## 💡 Лайфхаки для максимуму з безкоштовного плану

### 1. Використовуйте Pro Search стратегічно
Зберігайте 5 безкоштовних Pro Search для складних запитів. Прості питання відправляйте у Quick Search.

### 2. Чергуйте моделі
Коли ліміт GPT вичерпано — переключайтесь на Claude або Gemini. Вони теж доступні безкоштовно (з лімітами).

### 3. Spaces для документів
3 безкоштовних Spaces — створіть один для роботи, один для особистого, один для навчання. Завантажте туди PDF та використовуйте AI-пошук.

### 4. Focus режими замість Pro Search
Academic / Reddit / YouTube Focus режими доступні безкоштовно і значно покращують якість результатів без витрати Pro Search.

```
Безкоштовний трюк:
❌ [Pro Search] "Що думають підприємці про n8n?"
✅ [Reddit Focus] "n8n automation" → Реальні думки людей без витрати Pro Search
```

### 5. Збережіть треди
Perplexity зберігає всі ваші запити безкоштовно. Організуйте їх у колекції для швидкого доступу.

## 🏆 Підсумок

Perplexity Free — **найщедріший безкоштовний план** серед AI-пошуковиків. Більшість звичайних користувачів цілком задоволені Free.

Pro варто брати тим, хто **щодня** потребує глибокого аналізу, Deep Research звітів або генерації зображень — і тоді $20/міс повністю окупаються.
