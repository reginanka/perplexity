# Perplexity vs Gemini — порівняння

> Google Gemini та Perplexity AI — два потужні інструменти, що конкурують у сфері AI-пошуку. Але підходять вони для різних задач.

---

## 🎯 Концептуальна різниця

**Perplexity** — незалежний AI-пошуковик, що агрегує результати з усього інтернету та обов'язково показує джерела.

**Google Gemini** — AI-асистент від Google, інтегрований в екосистему Google (Gmail, Docs, Drive, Search). Gemini також вміє шукати в інтернеті через Google Search.

---

## 🔍 Де Perplexity краще

### 1. Незалежність від екосистеми
Perplexity не прив'язаний до Google. Він агрегує результати з різних джерел об'єктивніше — Google може просувати власні продукти та партнерів.

### 2. Focus режими для вузьких пошуків
Perplexity має режими Academic (наукові статті), Reddit (думки спільноти), YouTube. Gemini покладається на загальний Google Search.

### 3. Прозорість цитувань
Кожен факт у Perplexity має пронумероване посилання. В Gemini джерела є, але менш систематично представлені.

### 4. Spaces як база знань
Spaces — унікальна функція Perplexity для командної роботи з власними документами + AI-пошуком. У Gemini це реалізовано через Google Drive, що складніше налаштувати.

---

## 🌐 Де Gemini краще

### 1. Інтеграція з Google Workspace
Якщо ви активно використовуєте Gmail, Google Docs, Drive — Gemini прямо в цих додатках. Перепишіть листа, підсумуйте документ, знайдіть файл голосом.

```
✅ Gemini в Gmail: "Підсумуй цей ланцюжок листів і склади відповідь"
✅ Gemini в Docs: "Покращ цей абзац, зроби більш формальним"
✅ Gemini в Drive: "Знайди файл про маркетинговий план за Q1"
```

### 2. Мультимодальні можливості
Gemini 2.0 Flash та Gemini Ultra обробляють відео, аудіо, зображення значно краще. Він може дивитися YouTube-відео і відповідати на питання про нього.

### 3. Безкоштовний API
Gemini API має безкоштовний tier — для розробників це величезна перевага для тестування та невеликих проєктів.

```python
import google.generativeai as genai
# Безкоштовно до певних лімітів!
genai.configure(api_key="YOUR_KEY")
model = genai.GenerativeModel('gemini-2.0-flash')
response = model.generate_content("Що нового у світі AI?")
```

### 4. Контекстне вікно
Gemini 1.5 Pro та 2.0 мають контекстне вікно до **1 мільйону токенів** — можна завантажити цілу книгу або великий кодовий репозиторій.

## 💡 Практичні сценарії

| Задача | Кращий вибір |
|---|---|
| Швидкий веб-пошук з джерелами | ✅ Perplexity |
| Робота з Gmail/Docs/Drive | ✅ Gemini |
| Академічний ресерч | ✅ Perplexity (Academic mode) |
| Аналіз відео/аудіо | ✅ Gemini |
| Deep Research звіт | 🤝 Обидва |
| Безкоштовний API | ✅ Gemini |
| Незалежний пошук | ✅ Perplexity |
| Великий контекст (книга/код) | ✅ Gemini |

## 🔧 API для розробників

**Perplexity Sonar:**
```
sonar         — ~$1/1M tokens (з пошуком!)
sonar-pro     — ~$3/1M tokens
sonar-deep-research — дорожчий, глибокий аналіз
```

**Google Gemini API:**
```
gemini-2.0-flash — БЕЗКОШТОВНО (до лімітів)
gemini-1.5-pro   — $3.50/1M input tokens
gemini-2.0-pro   — преміум тариф
```

> 💡 Для Telegram-ботів або n8n автоматизацій — Gemini Flash безкоштовно дає відмінну якість. Для ботів з актуальним пошуком — Perplexity Sonar.

---

## 🏆 Кому що підходить

**Perplexity** — для незалежного дослідження, журналістів, підприємців, розробників що хочуть пошук у своїх продуктах.

**Gemini** — для тих, хто глибоко в екосистемі Google, потребує мультимодальних можливостей, або хоче безкоштовний API.
