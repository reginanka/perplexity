# 📊 Бенчмарки та реальні тести моделей у Perplexity

> Офіційні бенчмарки часто не відповідають реальному досвіду. Тут — практичні тести для типових задач.

---

## Важливе застереження

Бенчмарки — це орієнтир, не вирок. Модель з нижчим балом може бути кращою для **твоєї** конкретної задачі. Завжди тестуй сам.

---

## Тест 1: Написання коду (Python Telegram бот)

**Завдання:** Напиши aiogram 3.x хендлер для inline keyboard з колбеком

| Модель | Якість коду | Актуальність API | Пояснення |
|---|---|---|---|
| GPT-5 | ⭐⭐⭐⭐⭐ | ✅ aiogram 3.x | Детальні коментарі |
| Claude Sonnet | ⭐⭐⭐⭐⭐ | ✅ aiogram 3.x | Чистий стиль |
| Sonar Pro | ⭐⭐⭐ | ⚠️ може дати 2.x | Шукає приклади в мережі |
| Gemini 2.5 | ⭐⭐⭐⭐ | ✅ | Добре, але багатослівно |

**Переможець:** GPT-5 = Claude Sonnet (однаково добре)

## Тест 2: Актуальні дані (ціна Bitcoin сьогодні)
