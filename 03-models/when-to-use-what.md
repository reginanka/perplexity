# 🎯 Коли яку модель вибирати

> Правильний вибір моделі може кардинально покращити якість відповіді і зекономити час.

---

## Швидкий довідник

| Задача | Модель |
|---|---|
| Швидке питання з пошуком | Sonar (Auto) |
| Глибокий аналіз + пошук | Sonar Pro |
| Написання коду | GPT-5 або Claude Sonnet |
| Великий документ (200k+ токенів) | Sonar Pro або Gemini 2.5 Pro |
| Математика / логіка | o3 або sonar-reasoning-pro |
| Написання тексту, есе | Claude Sonnet / Opus |
| Новини та X/Twitter | Grok 3 |
| Мультимодальне (фото + текст) | Gemini 2.5 Pro або GPT-4o |
| Аналіз конкурентів (актуальний) | Sonar Pro + Deep Research |

---

## Порівняння за характеристиками

### Якість тексту і стиль письма
```
Claude Opus > Claude Sonnet > GPT-5 > Gemini > Sonar Pro
```

### Кодування
```
GPT-5 ≈ Claude Sonnet > o3 > Gemini 2.5 Pro > Sonar Pro
```

### Актуальність даних
```
Sonar Pro > Sonar > Grok 3 > GPT-5* > Claude* > Gemini*
(* без пошуку, з пам'яті тренування)
```

### Швидкість відповіді
```
Sonar Flash > GPT-4o mini > Gemini Flash > Sonar > GPT-4o > Claude Sonnet
```

### Розмір контекстного вікна
```
Gemini 2.5 Pro (1M) > Sonar Pro (200k) > Claude (200k) > GPT-5 (128k) > Sonar (128k)
```

## Практичні сценарії

### 📝 Написання статті/поста
```
Рекомендація: Claude Sonnet або GPT-5
Чому: найкращий стиль, структура, мінімум «роботизованості»
```

### 🐛 Debugging коду
```
Рекомендація: GPT-5 або Claude Sonnet
Чому: розуміють контекст помилок, дають конкретні виправлення
```

### 📊 Дослідження ринку
```
Рекомендація: Sonar Pro + Deep Research
Чому: актуальні дані + глибокий аналіз
```

### 🤖 Написання Telegram бота (Python)
```
Рекомендація: GPT-5 або Claude Sonnet
Чому: краще знають aiogram/python-telegram-bot патерни
```

### 📰 Моніторинг новин
```
Рекомендація: Sonar або Grok 3
Чому: завжди свіжі дані
```

---

## Хак: Перемикай моделі в межах одного чату

Perplexity дозволяє змінити модель прямо в процесі розмови. Наприклад:
1. Почни з Sonar — отримай актуальні дані
2. Перейди на Claude Sonnet — попроси гарно переписати
3. Перейди на o3 — перевір математику
