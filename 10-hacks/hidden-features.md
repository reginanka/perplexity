# Маловідомі можливості Perplexity

> Функції, про які не розказують у рекламі, але які суттєво підвищують продуктивність.

---

## 🔍 1. Пошук по конкретному сайту

Perplexity підтримує оператор `site:` прямо у запиті:

```
site:github.com n8n telegram bot examples
```

```
site:reddit.com/r/MachineLearning best RAG frameworks 2025
```

Працює в стандартному Web Focus режимі.

---

## 📎 2. Аналіз URL

Встав посилання у запит — Perplexity прочитає сторінку і відповість на питання:

```
https://docs.n8n.io/integrations/builtin/app-nodes/n8n-nodes-base.telegram/

Які тригери є у Telegram ноді n8n?
```

Працює з документацією, статтями, GitHub README.

> ⚠️ Не всі сторінки доступні (paywall, JS-only сайти).

---

## 📂 3. Аналіз PDF та файлів (Pro)

З Pro-планом можна завантажити:
- PDF документи
- CSV таблиці
- Зображення (скриншоти, схеми)

Потім ставити питання напряму по вмісту файлу.

---

## 🔗 4. Shareable Links

Кожен пошук в Perplexity має унікальне посилання:
- Відкрий будь-який результат пошуку
- Скопіюй URL з адресного рядка
- Поділись — отримувач побачить той самий результат з джерелами

Корисно для командної роботи та документації.

---

## 📚 5. Колекції (Collections)

Можна зберігати пошуки у тематичні папки:

1. Натисни кнопку збереження (bookmark) на результаті
2. Додай до нової або існуючої колекції
3. Колекції доступні в лівому меню

Зручно для довготривалих проектів і досліджень.

---

## 🖼️ 6. Генерація зображень

Perplexity Labs включає генерацію зображень через FLUX та інші моделі:

```
Намалюй схему архітектури RAG-системи з векторною базою даних
```

Доступно через лабораторні моделі (perplexity.ai/labs).

---

## 🌐 7. Мультимовний пошук

Perplexity шукає джерела на **мові запиту** і додатково на англійській:

```
Що таке агентні системи в ШІ?
```
→ Знаходить і українські, і англійські джерела, відповідає українською.

Для суто англійських результатів:
```
What are agentic AI systems? (respond in Ukrainian)
```

---

## ⌨️ 8. Клавіатурні скорочення

| Дія | Скорочення |
|---|---|
| Новий пошук | `Ctrl/Cmd + K` або `/` |
| Переключення Focus | `Alt + F` |
| Копіювати відповідь | кнопка в правому куті блоку |
| Відкрити джерело | натисни номер цитати `[1]` |

---

## 🤖 9. Perplexity Assistant (мобільний)

Мобільний застосунок має окремого **Assistant** — голосовий режим + агентні задачі:
- Голосові запити з розпізнаванням мови
- Нагадування і планування (обмежено)
- Інтеграція з камерою (сфотографуй і запитай)

---

## 🔄 10. Перегенерація з іншою моделлю

Якщо відповідь не задовольняє:
1. Натисни `⟳ Retry`
2. Вибери іншу модель зі списку
3. Отримай новий варіант без повторного набору запиту

Корисно порівняти відповіді Claude vs Sonar vs GPT на одне питання.
