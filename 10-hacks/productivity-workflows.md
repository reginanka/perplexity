# Воркфлоу для щоденної продуктивності з Perplexity

> Готові схеми використання Perplexity у реальних робочих процесах розробника та автоматизатора.

---

## ⚡ Воркфлоу 1: Ранковий дайджест новин

**Задача:** Бути в курсі AI/tech новин без трати часу.

**Запит (щодня):**
```
Що важливого сталося в AI та автоматизації за останні 24 години?
Фокус: нові моделі, оновлення інструментів, важливі анонси.
Формат: маржований список з посиланнями на джерела.
```

**Де використовувати:** Focus → Web, або Academic для наукових новин.

---

## ⚡ Воркфлоу 2: Ресерч перед розробкою

**Задача:** Перед написанням коду — знайти найкращий підхід.

**Послідовність запитів:**
```
1. "Які бібліотеки Python найкращі для [задача] у 2025 році?"
2. "Порівняй [бібліотека A] та [бібліотека B]: продуктивність, документація, підтримка"
3. "Покажи приклад мінімальної реалізації [задача] з [вибрана бібліотека]"
4. "Які найчастіші помилки при роботі з [бібліотека]?"
```

---

## ⚡ Воркфлоу 3: Дебагінг через Perplexity

**Задача:** Вирішити помилку без довгого гугління.

**Шаблон:**
```
Отримую помилку:
[вставити повний текст помилки]

Контекст:
- Мова: Python 3.11
- Бібліотека: [назва + версія]
- Що намагаюсь зробити: [опис]

Що може бути причиною і як виправити?
```

> 💡 Perplexity знаходить актуальні обговорення на GitHub Issues і Stack Overflow.

```
1. "Хто основні гравці на ринку [ніша] у 2025 році?"
2. "Які болі клієнтів у сфері [ніша] обговорюються на Reddit та форумах?"

```

**Focus для кроку 2:** Reddit — реальні відгуки без маркетингу.

## ⚡ Воркфлоу 6: Telegram-бот + Perplexity API (автоматизація)

**Задача:** Автоматичний щоденний дайджест у Telegram.

**Схема (n8n або Python):**
```
Cron (щодня 8:00)
  → Perplexity API (sonar-pro)
    → промпт: "AI новини за 24 години, коротко"
  → Telegram Bot API
    → відправити у канал/чат
```

**Python-скрипт (мінімальний):**
```python
import requests

api_key = "your_key"
response = requests.post(
    "https://api.perplexity.ai/chat/completions",
    headers={"Authorization": f"Bearer {api_key}"},
    json={
        "model": "sonar-pro",
        "messages": [{
            "role": "user",
            "content": "AI та автоматизація: топ-5 новин за останні 24 години. Коротко, з посиланнями."
        }]
    }
)
print(response.json()["choices"][0]["message"]["content"])
```
