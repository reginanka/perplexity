# Максимум з безкоштовного плану Perplexity

> Як вичавити 100% користі з Free-плану без жодної оплати.

---

## 🆓 Що є безкоштовно

| Функція | Free | Pro |
|---|---|---|
| Пошук (стандартний) | ✅ Необмежено | ✅ Необмежено |
| Pro Search | ⚠️ ~5 на день | ✅ Необмежено |
| Deep Research | ❌ | ✅ |
| GPT-4o / Claude | ❌ | ✅ |
| Завантаження файлів | ❌ | ✅ |
| Spaces | ❌ | ✅ |
| API доступ | ❌ | ✅ |

---

## 🛠️ Трюк 1: Зберігай Pro Search на важливе

Beзкоштовний план дає ~5 Pro Search запитів на добу.

**Стратегія:**
- Звичайний пошук — для швидких фактів і новин
- Pro Search — тільки для складних запитів з аналізом
- Нові 5 запитів поновлюються щодня о опівночі за UTC

---

## 🛠️ Трюк 2: Використовуй Focus-режими

Фільтри доступні безкоштовно і суттєво покращують якість:

- **Academic** — для наукових питань (PubMed, arXiv)
- **YouTube** — шукати відео замість тексту
- **Reddit** — думки реальних людей, не SEO-сміття
- **Web** — стандартний, але без шуму від Pinterest/Forbes

---

## 🛠️ Трюк 3: Ланцюг запитів (Chaining)

Замість одного великого запиту — серія коротких:

```
1. "Що таке векторні бази даних?" → отримуєш основу
2. "Порівняй Pinecone та Qdrant" → деталі
3. "Як підключити Qdrant до Python?" → практика
```

Кожен запит будує на попередньому в рамках однієї сесії.

---

## 🛠️ Трюк 4: Мобільний додаток — окремі ліміти

Ліміти Pro Search у веб-версії та мобільному додатку можуть оновлюватись незалежно. Перевір — іноді ліміт в застосунку ще не вичерпаний.

---

## 🛠️ Трюк 5: Безкоштовний API через партнерів

- **Vercel AI SDK** — безкоштовний тестовий доступ до Sonar
- **n8n Cloud** — вбудований Perplexity node (потребує власний API ключ, але є тріал)
- Студентські та освітні програми — перевір на [perplexity.ai/pro](https://perplexity.ai/pro)

---

## 💡 Підсумок стратегії

```
День → звичайний Web пошук для 95% задач
       ↓
       5 Pro Search → складний аналіз / порівняння
       ↓
       Focus режими → Academic / Reddit / YouTube
       ↓
       Ланцюг запитів у сесії → глибше без Pro
```
